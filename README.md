EEG Sentiment Analysis
Classifies human emotions into POSITIVE, NEGATIVE, and NEUTRAL using EEG signals from the DEAP dataset.


What This Project Does

Loads and cleans the dataset
Removes bad features, scales values, and reduces dimensions with PCA
Trains 7 models and compares their performance
Tunes the best model, builds ensemble models, and runs statistical tests
Analyzes errors and plots learning curves
